## ✨ 系统模块

### 1. 熙熙雅思主站（PHP）
- 听力工厂、口语工厂、阅读工厂、写作工厂、词汇工厂
- 雅思真题库（Cambrige4 等）
- 音频自动切片器、听力材料上传、管理
- 用户登录、学习进度追踪、访问统计

### 2. PDFDing（Docker）
- 专业的 PDF 文档管理系统（上传、阅读、搜索、标注）

### 3. 后端服务
- **Nginx**：80/443 端口（反向代理）
- **PHP 8.3-FPM**
- **MySQL**：数据库 `iagsv17`

```bash
